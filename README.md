# 多个js/css文件合并为1个
 合并多个js文件为1个 或  合并多个css文件为1个 ，使用gulp工具


## 使用
1. 请安装NPM相关包
`npm install` (建议用`cnpm install`)

2. 打开`gulpfile.js`按示例提示，自行按实际修改src文件

3. 编译合并指定文件
 `npm run build`
